1. 데이터 베이스 처리
	스키마 생성 -> 테이블 -> 기능 구현을 위한 sql 생성

2. 프로젝트 생성 : JRE, Runtime

3. 라이브러리 추가 : /WEB-INF/lib 에 추가
	데이터베이스 드라이버 추가. 
	JSTL 추가 (우리는  여기까지 추가 필요)
	파일 업로드 관련된것들도 추가
	필요시 JSON, XML 관련 라이브러리 추가.

4. 애플리케이션 초기화 : 
	서블릿 -> 데이터베이스 드라이버 로드

5. 기능 클래스 정의 
	Connection, Close, ...

6. 사용자 요청 기능들 구현
	- JSP(요청을 받는 기능) : jsp -> 1) 기능을 처리하는 service 찾아서 요청. 2) 결과데이터를 뷰로 전달 3) view 지정
	- Service Class(기능처리)
	- Domain (데이터 저장)
	- DAO (데이터 베이스 기능 처리)
	- JSP (응답을 표현 View)

	오늘까지는 요청을 처리하는 기능을 jsp로 만든다. 
	나중엔 요청 받고 분석하고 기능처리해주고 (서비스)
	결과 데이터 공유하고 뷰 지정하는 기능들 -> 서블릿으로 처리한다.
